## JS Moderno con Vite
Instrucciones para desplegar el proyecto en local:
1. Clonar el repositorio.
2. Ejecutar en la carpeta del proyecto  ```npm create vite@latest```.
3. Ejecutar ```npm run dev``` para ver el proyecto.
